#### **(Практическая работа 9-10) (Лаба 5\)**

#### **1\. Доработка логики врагов.**

* **Обычные мобы:** вся логика врагов должна описываться стейт машиной. Должны быть минимум 4 состояния \- Покой, Агрессия, Атака, Бегство. Должен быть применен подход \- машина состояний.  
* **Босс:** Должен быть добавлен босс. Босс должен иметь минимум 4 состояния \- Покой, Агрессия, Атака, Сильная атака.   
  **На доп. балл: минимум 7 состояний.**

**2\. Мирный режим.**

* **Обычные мобы:** не агрятся даже при атаке. Если у них мало хп \- убегают.  
* **Босс:** не агрится пока не ударят.

#### **(Практическая работа 9-10) (Лаба 6\)**

#### **1\. Вариативность мобов.**

* **Обычные мобы:** для моба ближнего и дальнего боя должны быть сделаны несколько видов оружия (минимум 2 для каждого класса). Необходимо реализовать спавнеры, которые можно свободно добавлять на карту.   
  **На доп. балл: случайные точки спавна или/и усиленные редкие мобы.**  
* **Босс:** Должен иметь два типа оружия/атаки и 4 стихии (лед, огонь, земля, эфир например), которые как-либо меняют процесс/результат атаки. Это может быть просто эффект (цвет/частицы). Важное условие \- на каждом виде оружия изменение должно выглядеть уникально. Например для оружия ближнего боя и ледяной стихии будет изменяться звук, а на атаку дальнего боя цвет частиц.   
  Босс изначально появляется с одним из типов оружия и стихии.
