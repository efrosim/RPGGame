### **(Практическая работа 5-6) (Лаба 3\)**

#### **1\. Доработки интерфейса**

* **Главное меню:** После запуска игры, игрок должен попасть в главное меню. Меню должно содержать две кнопки \- играть, настройки. При нажатии на кнопку играть, должна загружаться локация и начинаться игра. При нажатии на кнопку настройки, должно открыться меню настроек, в котором должен быть как минимум один пункт настроек (например громкость музыки).  
* **Игровое меню:** во время игры, игрок должен иметь возможность при нажатии на кнопку esc выйти в игровое меню, в котором должно быть три кнопки \- главное меню, сохранить и загрузить сохранение.

**2\. Реализация паттернов и архитектурных принципов:**

* В проекте явно должны быть реализован паттерн Entrypoint. Паттерн должен быть применен как для игры в целом (создаются общие сервисы, например \- сервис аудио, сервис сохранений), так и для каждой сцены в частности (настраиваются объекты на сцене, прокидываются реализации сервисов, настраиваются события и т.д.)  
* Проект должен быть выстроен сверху вниз, где сверху находятся конкретные реализации (конкретный сервис сохранения в файл или бд, конкретный сервис проигрывания музыки), а спускаясь вниз мы видим абстракции уровня сцен и абстракции уровня геймплея.  
* **На доп балл.** Реализовать работу с интерфейсом через MVC подход (и постараться не делать контроллер монобехом)
